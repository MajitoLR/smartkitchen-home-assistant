# 🍳 SmartKitchen Home Assistant

SmartKitchen es una aplicación web multimodal de cocina inteligente desarrollada con Streamlit, ESP32 y Wokwi.

El proyecto busca mejorar la experiencia en la cocina mediante automatización inteligente, monitoreo de sensores y asistencia interactiva.

---

# 🚀 Funciones principales

## 📖 Asistente de recetas

- búsqueda de recetas
- ingredientes
- instrucciones paso a paso
- temporizador
- comandos simulados de voz

---

## 🔥 Monitor inteligente de cocina

- monitoreo de temperatura
- alertas visuales
- sensores simulados
- estado de dispositivos
- dashboard interactivo

---

# 🧠 Multimodalidad

El proyecto utiliza diferentes formas de interacción:

| Modalidad | Uso |
|---|---|
| Visual | Dashboard y alertas |
| Texto | Búsqueda de recetas |
| Audio | Alarmas |
| Sensores físicos | Temperatura |
| Botones | Interacción |
| Voz simulada | Comandos |

# 🎯 Objetivo

Crear una experiencia de cocina inteligente que combine:

- automatización
- sensores físicos
- interfaces multimodales
- interacción digital

para mejorar la seguridad y comodidad del usuario en la cocina.
